# Chá Rifa do Bebê Miguel Gabriel

Aplicativo web para escolher números (1 a 100) online.

- Pix: R$ 45,00
- 100 números
- Fralda automática por faixa (P/M/G)
- Painel admin para confirmar pagamento (status "paid")

## Variáveis de ambiente (na Vercel)
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY
- NEXT_PUBLIC_EVENT_ID
- ADMIN_PASSWORD
